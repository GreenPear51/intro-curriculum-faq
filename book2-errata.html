「Webプログラミングが面白いほどわかる本」への補足

このページでは 書籍 「Webプログラミングが面白いほどわかる本 環境構築からWebサービスの作成まで、はじめからていねいに」の出版後に変更された点などをまとめています。

RLogin の画面が英語から日本語になった

Windows で 仮想環境上の Ubuntu に SSH 接続する場合に使用する RLogin は、書籍出版時の段階で設定画面の一部が英語でしたが、現在は日本語になっています。
機能に違いはありませんので、以下の通り読み替えて学習を進めてください。

英語と日本語の対照

  • Server Address → ホスト名（サーバーIPアドレス）
  • Socket Port → TCPポート
  • User Name → ログインユーザ名
  • SSH Idenitiy Key → SSH 認証鍵
